# DSA Repository 🚀

A collection of Data Structures and Algorithms problems solved in Java. This repository contains coding practice questions, algorithm implementations, and problem-solving approaches commonly asked in coding interviews and technical assessments.

---

## 📚 Topics Covered

### Arrays

* Move All Zeros to End
* Two Sum
* Maximum Subarray
* Merge Sorted Arrays
* Remove Duplicates

### Strings

* Reverse String
* Palindrome Check
* Anagram Check
* Longest Common Prefix

### Searching

* Linear Search
* Binary Search
* Search in Rotated Sorted Array

### Sorting

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### Linked List

* Reverse Linked List
* Detect Cycle
* Middle of Linked List

### Stack & Queue

* Valid Parentheses
* Implement Stack
* Implement Queue

### Recursion

* Factorial
* Fibonacci Series
* Recursive Array Problems

### Sliding Window

* Maximum Sum Subarray
* Fixed Window Problems
* Variable Window Problems
